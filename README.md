# TopFix 🔧

TopFix is a multi-page service-based website designed for a laptop and desktop repair company.  
It provides doorstep repair services, online product sales, service booking, and location-based support.

## 🚀 Features
- Doorstep laptop & desktop repair services
- Product listing and online shop UI
- Sell used devices marketplace
- Location-based service centers
- Contact & service request forms
- Login page UI
- Responsive navigation across all pages

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript 

## 📂 Pages Included
- Home
- Services
- Products
- Sales (Sell Products)
- Locations
- Contact
- Login
